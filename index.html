French University Research

Track documents, deadlines, departments, and professors.

Use this repository to build a clean application database for French universities, compare departments, track required documents, and save ResearchGate and Google Scholar links with citation counts for each professor you shortlist.

Open README Getting Started Collection Guide

Documents

Track passports, transcripts, CVs, motivation letters, recommendation letters, and research proposals.

Deadlines

Record application openings, closings, interviews, and scholarship milestones by university and program.

Research Fit

Compare departments, labs, and professor research areas in one filterable master sheet.

Profiles

Save ResearchGate, Google Scholar, and university profile URLs alongside citation counts and notes.

Update the CSV files in data/, then run npm run build:xlsx to refresh the Excel workbooks.
